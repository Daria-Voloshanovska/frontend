Lesson 5: Position

Значение свойства позиционирования

  • static: свойство position по-умолчанию
  • fixed: элемент вырывается из потока элементов на странице и становится невидимым для скрола
  • sticky: элемент приклеивается при скроле сохраняя за собой место в выбраной точке, вырывается из потока
  • relative:двиигает элемент относительно своего места не вырывая из потока - чаще всего используется в связке с absolute (позиционирование родителя)
  • absolut: позиционируем элемент относительно body или первого спозиционированного родителя

Lesson_04: Flexbox

Flexbox - это гибкий инструмент для позиционнирования элементов в CSS

  • Чтобы "включить" flex отображение в контейнере нужно задать свойство display:flex
  • justify-content - свойство выравнивания по главной оси
  • align-items - выравнивания по поперечной оси
  • flex-direction - свойство меняющие оси местами
  • gab - промежуток между элементами в flex контейнере
